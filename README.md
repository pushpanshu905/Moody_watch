# 🎬 MoodWatch

A real-time watch party app built using Node.js, Express, and Socket.IO.

## 🚀 Live Demo
👉 [View Project](https://your-live-link-here)


---

## ✨ Features
- Create & join rooms
- Watch YouTube videos together
- Live chat
- Emoji reactions
- Max 4 users per room

---

## 🛠️ Tech Stack
- Node.js
- Express
- Socket.IO
- HTML, CSS, JavaScript
